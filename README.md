# 💌 Digital Wedding Invitation

Welcome to the repository of my digital wedding invitation landing page! This project is a responsive, elegant, and interactive web-based invitation designed to share our special day with friends and family.

### ✨ Features
* **Responsive Design:** Optimized for both mobile and desktop viewing.
* **Interactive Details:** Countdown timer, digital RSVP form, and integrated maps for the venue.
* **Audio Background:** Soft background music to enhance the invitation experience.

### 🛠️ Built With
* HTML5 
* CSS3 
* JavaScript
